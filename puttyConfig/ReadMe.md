# 文件说明

`putty配置注册表.reg`是window下的注册脚本文件，此注册表内包含了对putty文本字体,以及配色的修改从而达到画面和颜色都美观的目的

# putty字体

字体设为`Fira Mono`,字形为粗体，大小为14号（如果你的电脑屏幕较小可以设为12号）



# 安装方法

双击REG文件即可将其中的数据导入到注册表当中。
